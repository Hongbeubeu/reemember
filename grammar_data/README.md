## How to import

In Reemember, go to the **Grammar** tab → click **Import** → select any `.json` file.
Each file is a JSON array `[{ ... }]` containing one grammar document with exercises.

See `GRAMMAR_DATA_FORMAT.md` in the project root for the full schema and exercise type reference.
